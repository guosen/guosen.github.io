简介

Bolts是异步执行的第三方库，顺序结构。采用嵌套的方式，往上回调

用法

1定义一个任务

～～
private Task uploadImages(final Post post, List images) {
Task ret = Task.forResult(null);
for (final Image image : images) {
ret = ret.onSuccessTask(new Continuation>() {
@Override
public Task then(Task task) throws Exception {
if (post.mPublishingStatus == PublishingStatus.CANCELED) {
//取消发布。
return Task.forError(new CancelledException());
}
return uploadImage(image);
}
});
}
return ret;
}
